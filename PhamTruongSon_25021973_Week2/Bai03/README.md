Bài 2.3: The swap trick
Yêu cầu:
- Tạo class NumberWrapper chứa một số nguyên int value. 
- Tạo setter/getter tương ứng.
- Viết hàm swap(NumberWrapper a, NumberWrapper b) thực hiện hoán đổi vị trí 2 tham số a và b (gán temp = a; a = b; b = temp;).
- Viết hàm main:
- Tạo 2 đối tượng n1 (value=5) và n2 (value=10).

Gọi swap(n1, n2).
In giá trị của n1, n2 sau khi gọi hàm.
